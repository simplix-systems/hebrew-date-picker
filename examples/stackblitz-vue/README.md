# Hebrew Date Picker — Vue demo (StackBlitz-ready)

A minimal Vue 3 + Vite project showing `@simplix-systems/hebrew-date-picker`:
an inline single-date picker, a range picker with the quick-range presets
sidebar, and a popup input field.

**Open it instantly on StackBlitz:**

https://stackblitz.com/github/simplix-systems/hebrew-date-picker/tree/main/examples/stackblitz-vue

Or run locally:

```bash
npm install
npm run dev
```
